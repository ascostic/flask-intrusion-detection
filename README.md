# Flask Intrusion Detection System

## Overview
A fully automated login monitoring and intrusion detection system built using Python and Flask.

The system automatically detects brute-force attacks, enforces account lockout policies, blocks suspicious IP addresses, and generates automated security logs without manual intervention.

## Core Features (Planned)
- Secure User Authentication
- Password Hashing (bcrypt)
- IP-Based Rate Limiting
- Automated Brute-Force Detection
- Account Lock Mechanism
- Suspicious Login Detection
- Background Log Monitoring (Scheduler)
- Automated Security Reporting

## Tech Stack
- Python
- Flask
- MySQL
- APScheduler
